    This python package generates pore structures to be used in molecular simulations. Visit the porcess page for an overview of the programs operating principle. Additionally tutorials for generating molecules and pores are provided.

    Check out an examplary workflow for using the PoreMS package to create a pore system and run molecular dynamics simulation using Gromacs.
