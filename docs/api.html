    API reference¶

    Generation¶

    molecule.Molecule([name, short, inp]) This class defines a molecule object, which is basically a data matrix that contains atom information of positions and type.
    pore.Pore(size, diam, drill[, res, vs, …]) This class is an extension of the molecule class porems.molecule.Molecule and the core class for creating silicon pores.
    store.Store(molecule[, link]) This class converts a molecule object to a structure file of different possible formats.

    Essential Molecules¶

    essentials.Alkane(length[, name, short, is_h]) Using this class linear alkane molecules can be easily constructed with the only input of the carbon atom number.
    essentials.Alcohol(length[, name, short, is_h]) Using this class linear alcohol molecules can be easily constructed with the only input of the carbon atom number.
    essentials.Ketone(length, pos[, name, …]) Using this class linear ketone molecules can be easily constructed with the only input of the carbon atom number and oxygene position.
    essentials.TMS([name, short, charge, …]) This class defines trimethylsilyl (TMS) bound to a silicon-grid.

    Pore Optimization¶

    bonding.Bonding(verlet) The aim of this class is preserving all information of the silicon grid bonds.
    verlet.Verlet(mol, size, is_pbc) This class converts the molecule box into verlet lists.
