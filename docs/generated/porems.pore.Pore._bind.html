    Pore._bind¶

    Pore._bind(self)¶

    Create binding site matrix. This matrix \(\boldsymbol{B}\) is mostly generated by the bonding class poresms.bonding.Bonding

    \[\begin{split}\boldsymbol{B}=\begin{bmatrix} o_0&s_0&p_0&u_0&t_0&g_0&m_0\\ o_1&s_1&p_1&u_1&t_1&g_1&m_1\\ \vdots\\ o_b&s_b&p_b&u_b&t_0&g_b&m_b\\ \end{bmatrix}\end{split}\]

    with binding site number \(b\) and entries

    1. Oxygen atom id \(o_{k=0,\dots,b}\)
    2. Silica atom id \(s_k\)
    3. List of pointers \(p_k\) to binding sites that are in proximity
    4. State \(u_k\): available - 0, used - 1, is in proximity - 2
    5. Type \(t_k\): inside the pore - 0, outsite the pore - 1
    6. Pointer \(g_k\) to geminal binding site
    7. Molecule position \(m_k\) in the global molecule list
