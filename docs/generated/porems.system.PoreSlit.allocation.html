    PoreSlit.allocation¶

    PoreSlit.allocation()¶

    Calculate molecule allocation on the surface. Using interior and exterior surfaces, the allocation rates can be determined by counting the number of used molecules on the surfaces.

    Using the conversion function porems.utils.mols_to_mumol_m2(), the number of molecules is converted to concentration in \(\frac{\mu\text{mol}}{\text{m}^2}\).

    Returns:
    allocdictionary

    Dictionary containing the surface allocation of all molecules in number of molecules, \(\frac{\text{mols}}{\text{nm}^2}\) and \(\frac{\mu\text{mol}}{\text{m}^2}\)
