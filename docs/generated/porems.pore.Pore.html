    Pore¶

    class porems.pore.Pore(size, diam, drill, res=1, vs=0.4, proxi_dist=0.5, is_pbc=True, is_time=False)¶

    This class is an extension of the molecule class porems.molecule.Molecule and the core class for creating silicon pores. Note that this class can therefore use all methods of the Molecule class.

    At the beginning a silicon-oxygen grid is generated from the smallest possible grid unit by duplication. Using the verlet list porems.verlet.Verlet and bonding classes porems.bonding.Bonding, this silicon grid is prepared rapidly and binding sites are provided.

    Here methods are provided for adding molecule groups to the surface of the generated pore. Furthermore, a method is added to fill all unused binding sites with silanol and siloxane groups.

    All bound molecules are added to a global molecule list, in order to preserve individual molecule properties.

    Parameters:
    size : list, float

    Size of the silicon-oxygen-grid, can be a float, if same size in all dimensions

    diam : float

    Pore diameter

    drill : string

    Axis for the pore drilling

    res : float, optional

    Value to extend and translate the pore by on the drill side creating reservoirs

    vs : float, optional

    Verlet-list box size

    proxi_dist : float

    Minimal distance between two groups attached to the surface

    is_pbc : bool, optional

    True if periodic boundary conditions are needed

    is_time : bool, optional

    True to print the used time for the single steps

    Examples

    Following example generates a pore with TMS molecules on the inside and outside surface

    from porems.pore import Pore
from porems.essentials import TMS

pore = Pore([10, 10, 10], 6, "z", 5.5)

pore.attach(TMS(), [0, 1], [1, 2], 0, 4.3, inp="molar")
pore.attach(TMS(), [0, 1], [1, 2], 1, 60, inp="percent")

pore.finalize()

    Private Methods - Building

    _block(self, dim, block) Recursively duplicate and translate a given molecule block in all given dimensions.
    _foundation(self) In this method the smallest possible grid unit is created in a way, that duplication is simply done by copying the molecule without the need of deleting atoms in the end.
    _orientation(self) Rotate pore orientation, so that the specified drill axis becomes the z-axis.

    Private Methods - Binding Sites

    _add_mol_list(self, mol_list) Add given molecule list to global molecule list and add the position in this list to the corresponding binding site.
    _bind(self) Create binding site matrix.
    _close(self, sites[, site_list]) This function closes the specified site by setting the used value to 1 and the used value of all sites in proximity to 2 in the given site list.
    _proxi(self) Using verlet lists, search for binding sites that are in proximity, by looking for oxygen atoms that are near each other.
    _random(self, site_type, rate, inp, counter) Get a random binding site list for a given percentage of the available binding sites.

    Private Methods - Final Edits

    _calc_props(self) This function calculates all necessary properties of the system and returns a dictionary.
    _connect(self) Concatenate two geminal molecules to one molecule.
    _excess(self[, is_rand]) Adjust grid atoms charges to compensate for excess charges on the pore.
    _objectify(self[, data]) Move all remaining grid silicon and oxygen atoms to individual molecules for writing the structure file and add these molecules to the global molecule list.
    _objectify_parallel(self) Parallelized function _objectify().
    _overlap(self) Method for checking of silicon or oxygen atoms are overlapping using verlet lists.
    _position(self) Translate the pore into position and consider the periodic repeat gap.
    _silanol(self[, sites]) Convert the remaining binding sites to silanol groups using function attach().
    _silanol_parallel(self) Parallelized function _silanol().
    _sort(self) Sort molecules in order to prevent multiple molecule appearances in the structure file.

    Public Methods - Surface Attachment

    attach(self, mol, si_o, orient, site_type, rate) Add a molecule to the pore binding site.
    attach_dual(self, mol, si_o, orient, rate[, …]) Add a molecule to two pore binding sites.
    siloxane(self, rate[, inp, counter]) Add siloxane bridges to the pore by selecting two binding sites in proximity removing both oxygens and placing one at centre of the two removed atoms.
    special(self, mol, si_o, orient, num, symmetry) Add a molecule in a specified orientation with a specific amount.

    Public Methods - Completion

    finalize(self[, is_rand]) Finalize pore by filling empty bond with silanol groups, connecting geminal molecules, removing marked atoms, converting the grid to individual molecules and finally moving the pore into position.
    table_props(self[, decimals]) This functions creates readable pandas DataFrames of all properties data.

    Getter Methods

    get_center(self) Return the pore focal point.
    get_diam(self) Return the pore diameter.
    get_drill(self) Return the drilling axis.
    get_gap(self) Return the gap vector.
    get_props(self) Return the final pore properties.
    get_q(self) Return the adjusted grid charges.
    get_reservoir(self) Return the reservoir length.
    get_size(self) Return the pore size.
    get_time(self) Return the total creation time.
